# 🪐 SpaceOver: Space Exploration for Visually Impaired 🚀

</div>

시각작애인 유저들이 스스로 우주를 탐색하고 태양계의 천체들을 찾을 수 있는 천문학적 경험을 제공합니다. <br/>
SpaceOver provides an astronomical experience for BVI(blind and visually impaired) users to independently navigate through space and locate the planets of the solar system.
